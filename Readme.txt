Education Institute Management System
This Java project is an education institute management system that allows students to enroll in various courses offered by the institute.

Project Description
The Education Institute Management System provides a platform for students to explore available courses, view course details, and enroll in their desired courses. The system supports multiple courses and offers various discounts and special offers.

Project Structure
The project is structured into the following packages:

com.education.classcourse: Contains classes related to course details.
com.education.educationinstitute: Contains classes for managing the education institute.
com.education.main: Contains the main class EducationInstituteApp to run the application.
com.education.offers: Contains classes for managing offers and discounts.
com.education.student: Contains classes related to student enrollment and management.

How to Run
To run the project locally, follow these steps:

Clone the repository to your local machine.
Open the project in your preferred Java IDE.
Compile and run the EducationInstituteApp class.

Usage
Upon running the application, users are prompted to select a course from the available options: C++, JAVA, PYTHON, or ORACLE. Users can then enroll in their desired course and view available offers and discounts.
